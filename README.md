# Annihilation Engine X Public Issue Tracker

This repository is the public issue tracker and roadmap for Annihilation Engine X editor tooling.

Tracked projects include:

- AEX Unit Editor
- AEX Map Editor

The core AEX engine and proprietary runtime remain private.

This repository exists for:
- Bug reports
- Feature requests
- Usability feedback
- Public roadmap tracking
- Community discussion

Please do not post:
- Proprietary source code
- Credentials or secrets
- Security-sensitive implementation details
- Paid or private game assets
